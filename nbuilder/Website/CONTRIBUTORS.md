# For Contributors

## Development Environment Requirements

1. Install [chocolatey] into your powershell environment
1. Install Silverlight 4
```powershell
choco install -y silverlight4sdk
```

## How to Build from the command line

```powershell
.\Invoke-Build.ps1
```
